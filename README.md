#  Tshepo Sepatake — Personal Portfolio

A personal portfolio website built with **HTML** and **CSS**, showcasing my skills, projects, and contact information as a Computer Science student with a strong interest in Software Development.



 # Live Preview

 Open `Home.html` in your browser or run with Live Server on VS Code.


# Project Structure

```
Portfolio/
├── Home.html
├── style.css
└── assets/
    └── images/
```

---

# Sections

 **Home** — Hero section with a welcome message, profile photo, and social links (GitHub, LinkedIn, X)
 **About Me** — Brief bio describing my background, interests, and current learning focus
**Skills** — Visual progress bars for Front-End (HTML, CSS, JavaScript) and Back-End (Java, Spring Boot, SQL) skills, plus Soft Skills
 **Projects** — Cards showcasing three projects with images and demo links:
  - Authentication Login Form
  - Marks Capturing System
  - Stadium Booking System
 **Contacts** — Contact form with Name, Email, WhatsApp Number, and Message fields



# Built With

- HTML5
- CSS3 (Flexbox, Grid, Gradients)
